Welcome to Movie-Recommender

This is my attempt at using what I learned in two recent Spark courses I took on EDx and put them into use in a production system. In this case I used the data from MovieLense to recommend movies based on user ratings. Ratings were created by creating cosine simlilarity scores for users against each other. For the most similar users in the dataset each movie they rated is then multiplied by their cosine similarity score to give the top rated movies for the user. This is then stored as JSON in a Postgres database and displayed to the user.

A few notes:

  • Currently I'm just running the scores against the small MovieLense dataset so my amazon bills on this stay near zero. If for some reason you want me to run your scores against the large dataset let me know.
  • If you don't want to user your email address when you sign up you can put in a username, or whatever you want for that field, I'm not error checking it.
  • Sorry for the generic bootstrap design, my goal was to spend no time on design ;).

If you're interested in hearing more about how I built it, reach out, I'd love to chat. You can find out more about me at camlinke.com

Some of libraries I used

  • Spark - Used PySpark to create the recommendations
  • Flask - Light python web framework
  • SQLAlchemy - ORM
  • Bcrypt - Encrypt passwords
  • Delorean - Eliminated datetime headaches
  • Flask-Login - Login/Out for Flask
  • Flask-Migrate - Easy migrations for Flask and SQLAlchemy using Alembic
  • Flask-WTF - Forms made easy for Flask
  • Jinja2 - Templating engine, comes built in with Flask.
  • RQ - Job queue for generating recommendations in background
  • Redis
